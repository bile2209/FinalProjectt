Responsive Design ~25 pts

Section 06

Four Extras ~25 pts

Extra  01
CSS Grid Layout
The Projects section uses a fully responsive CSS Grid with auto-fit and minmax(), so cards reflow automatically from 3 columns on desktop down to 1 on mobile without any media queries on the grid itself. 
.projects-grid {
  display:               grid;
  grid-template-columns:
    repeat(auto-fit, minmax(260px, 1fr));
  gap:                   var(--space-md);
}
Extra  02
CSS Custom Properties
The entire site's color palette, typography, spacing, and shadow system is defined once in :root as custom properties. Changing a single token updates every component simultaneously, making the design fully maintainable. 
/* Accent color change updates everywhere */
:root {
  --color-accent: #e94560;
}
.btn-primary {
  background: var(--color-accent);
}
.tag {
  color: var(--color-accent);
}
Extra  03
Scroll-Triggered Animations
Sections and project cards animate into view as the user scrolls down, using the native IntersectionObserver API. No external library — pure JavaScript with a CSS transition on opacity and transform. 
// scroll-animations.js
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting)
      entry.target.classList.add('visible');
  });
}, { threshold: 0.15 });

document.querySelectorAll('.reveal')
        .forEach(el => observer.observe(el));
Extra  04
Image Gallery / Lightbox
Clicking any gallery thumbnail opens a full-screen lightbox overlay with the enlarged image and a close button. Built in vanilla JavaScript — no plugins or libraries. Keyboard Escape also closes the lightbox. 
// lightbox.js
const lb  = document.getElementById('lightbox');
const img = lb.querySelector('img');

document.querySelectorAll('.gallery-item')
  .forEach(el => el.addEventListener('click', () => {
    img.src = el.dataset.full;
    lb.classList.add('active');
  }));

document.addEventListener('keydown', e => {
  if (e.key === 'Escape')
    lb.classList.remove('active');
});
